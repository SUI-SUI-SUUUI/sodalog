# そだログ

そだログは、LINEから園芸作業の記録を送るだけで、GoogleスプレッドシートとGoogle Driveに自動整理する園芸記録管理アシスタントです。

## 概要

庭仕事中にパソコンを開かなくても、スマホのLINEから作業内容を送るだけで、日付・場所・植物名・作業内容を記録できます。

番号を選びながら入力する方法と、1行テキストで送る方法の両方に対応しています。記録後に画像を送ると、Google Driveへ保存し、スプレッドシートの該当行へURLを追記します。

## 解決したい課題

園芸作業の記録は、写真・日付・場所・植物名・作業内容が分散しやすく、あとから振り返りにくいという課題があります。

そだログでは、普段使っているLINEを入口にすることで、庭仕事の流れを止めずに記録を残せる仕組みを作ります。

## 入力方法

### 番号選択で記録する

「記録する」と送ると、作業日 → 育成拠点 → 場所 → 詳細場所 → 植物名 → 作業内容 → メモの順で入力できます。入力途中は「中止」でキャンセルできます。

### 1行テキストで記録する

次の形式でも記録できます。

```text
2026/07/13_北側_中央_つつじ_剪定
```

形式：`作業日_場所_詳細場所_植物名_作業内容`

### その他のコマンド

| 送信テキスト               | 動作                     |
| -------------------------- | ------------------------ |
| 記録する / 記録 / 園芸記録 | 番号選択による記録を開始 |
| 最近の記録 / 最近 / 履歴   | 直近の記録を表示         |
| 使い方 / ヘルプ / メニュー | 操作方法を表示           |
| 中止 / キャンセル          | 入力途中の記録を中止     |

## 現在できること

- LINEからテキスト・画像を受信（Webhook）
- 番号選択による園芸記録入力（セッション管理）
- 1行テキスト形式での記録と入力検証
- Googleスプレッドシートへの記録保存（`garden_log` シート）
- 記録完了メッセージのLINE返信
- 入力形式エラー時の案内返信
- 使い方・最近の記録の表示
- LINEから送信した画像をGoogle Driveへ保存
- 保存した画像URL・DriveフォルダURLをスプレッドシートへ記録
- 場所 / 詳細場所 / 年ごとのDriveフォルダ自動作成
- LINEリッチメニューの作成・設定（GASから実行）

## コード構成

Google Apps Script のコードは `gas/` 配下で機能別に分割しています。

| ファイル               | 役割                                                        |
| ---------------------- | ----------------------------------------------------------- |
| `00_Config.gs`         | 定数・シート名・ヘッダー定義                                |
| `01_Webhook.gs`        | Webhook受信（`doPost` / `doGet`）、テキスト・画像の振り分け |
| `02_RecordFlow.gs`     | 番号選択による園芸記録入力フロー                            |
| `03_ValidationDate.gs` | 1行テキストの検証・日付処理                                 |
| `04_Image.gs`          | 画像保存・Driveフォルダ管理                                 |
| `05_HelpHistory.gs`    | 使い方・最近の記録                                          |
| `06_SheetsSessions.gs` | スプレッドシート操作・ユーザーセッション管理                |
| `07_LineUtilities.gs`  | LINE返信・JSON応答・デバッグログ                            |
| `08_RichMenu.gs`       | リッチメニューの作成・設定                                  |
| `99_Tests.gs`          | 動作確認用テスト                                            |
