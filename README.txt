あなたには、これから「My Garden Lab」というアプリ開発を手伝ってもらいます。

このアプリは、園芸作業の記録をLINEから簡単に残せる「園芸記録管理アシスタント」です。

目的：
庭仕事中に、パソコンを開かなくても、LINEから作業内容や写真を送るだけで、GoogleスプレッドシートとGoogle Driveに自動整理できる仕組みを作る。

主な機能：
1. LINEから作業内容を送信する
2. LINEから画像を送信する
3. Google Apps ScriptでLINE Webhookを受け取る
4. 作業内容をGoogleスプレッドシートに1行追加する
5. 画像をGoogle Driveに保存する
6. Drive内に「年 / 月 / 場所 / 詳細場所」の階層フォルダを自動作成する
7. 記録完了後、LINEに返信する

想定する入力例：
2026/04/11_北側_中央_つつじ_剪定

保存したい情報：
- 記録日時
- 作業日
- 場所
- 詳細場所
- 植物名
- 作業内容
- メモ
- 画像URL
- DriveフォルダURL

使用予定の技術：
- LINE Messaging API
- Google Apps Script
- Google Spreadsheet
- Google Drive
- GitHub

開発方針：
まずはMVPとして、LINEからテキストを送るとスプレッドシートに記録されるところまで作る。
その後、Driveフォルダ作成、画像保存、LINE返信、リッチメニューの順で進める。

勝手に高度な構成にせず、まずはGoogle Apps Script中心の軽量構成で進めたいです。
